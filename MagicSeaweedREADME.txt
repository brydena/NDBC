MagicSeaweed.py README
Andrew Bryden 17/02/2019

This command line program takes retrieves data from https://www.ndbc.noaa.gov/data/latest_obs/latest_obs.txt
and accepts input of a latitude and longitude and returns any stations with data in the last 3 hours and within
100 miles of the given latitude/longitude.

The library 'requests' will be required for this to run.
